# PRO
Programación (1DAW - IES Puerto de la Cruz)

## [UT1: Introducción a la Programación](./ut1/README.md)
Unidad sobre los conceptos básicos de la programación, lenguajes de programación y la historia de Python. / Unit about the basic concepts of programming, programming languages, and the history of Python.

## [UT2: Tipos de Datos](./ut2/README.md)
Unidad enfocada en los fundamentos de Python, incluyendo variables, números y operaciones con cadenas de texto. / Unit focused on Python fundamentals, including variables, numbers, and string operations.

## [UT3: Control de Flujo](./ut3/README.md)
Unidad dedicada a las estructuras de control de flujo en Python, abarcando usos de condicionales y bucles. / Unit dedicated to control flow structures in Python, covering the use of conditionals and loops.

## [UT4: Estructura de Datos (I)](./ut4/README.md)
Unidad sobre la creación, manipulación y métodos relacionados con las listas y estructuras de datos. / Unit about the creation, manipulation, and methods related to lists and data structures.

## [UT5: Estructura de Datos (II)](./ut5/README.md)
Unidad centrada en trabajar con colecciones clave-valor (diccionarios) y la manipulación de ficheros. / Unit focused on working with key-value collections (dictionaries) and file manipulation.

## [UT6: Funciones](./ut6/README.md)
Unidad que aborda funciones avanzadas, recursividad, decoradores y generadores en Python. / Unit covering advanced functions, recursion, decorators, and generators in Python.

## [UT7: Programación Orientada a Objetos](./ut7/README.md)
Unidad centrada en los conceptos de clases, objetos, métodos, herencia y encapsulamiento. / Unit focused on the concepts of classes, objects, methods, inheritance, and encapsulation.

## [UT8: Librerías](./ut8/README.md)
Unidad dedicada al uso de librerías, manejo de validaciones y expresiones regulares. / Unit dedicated to library usage, validation handling, and regular expressions.
